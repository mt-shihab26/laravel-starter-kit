# laravel-starter-kit

My Laravel + Inertia + React Starter Kit

## Requirements

- PHP 8.5+
- Composer
- Node.js
- Bun

## Installation

```bash
composer run setup
```

This will install dependencies, configure the environment, generate an app key, run migrations, and install frontend packages.

## Development

```bash
composer run dev
```

Starts the Laravel server, queue worker, log viewer, and Vite dev server concurrently.

## Testing

```bash
composer run test
```
